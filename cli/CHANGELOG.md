# Changelog

## 1.0.0

- Initial release
- CLI commands: `init`, `add`, `list`, `diff`
- 9 built-in widgets: button, card, input, badge, skeleton, separator, label, dialog, toast
- Design token system (`KineticTokens`) copied as source — fully editable
- Registry hosted on GitHub raw files
- Recursive component dependency resolution
- Auto-detect pubspec indentation (2-space and 4-space)
- Skip existing theme files on re-init (idempotent)
